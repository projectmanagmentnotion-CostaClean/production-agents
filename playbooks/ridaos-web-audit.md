# Ridaos Web Audit Playbook

## Objetivo

Auditar y corregir webs o ecommerce de Ridaos Print con criterio de producción, conversión y diseño senior.

## Áreas

- UX/UI comercial
- configuradores de producto
- mobile-first
- formularios y WhatsApp
- SEO local
- performance de imágenes
- accesibilidad
- copy y CTAs
- checkout o contacto
- diseño visual premium urbano

## Prompt base

```text
Ejecuta un Full Product Audit and Correction Pass usando production-agents.

Proyecto: Ridaos Print
Tipo: web/ecommerce/landing
Objetivo: mejorar conversión, claridad, diseño profesional, responsive y performance.

No tocar pagos/producción sin autorización.
No rediseñar todo sin necesidad.
Corregir P0/P1/P2 reales.
Validar, commit/push y entregar rollback.
```
